# dns-sync Wiki

Welcome to the **dns-sync** documentation.

> **Note:** This directory (`docs/wiki/`) is a placeholder for the GitHub Wiki submodule.
> After creating the GitHub repository, run:
>
> ```bash
> # Remove placeholder
> rm -rf docs/wiki
>
> # Add the wiki as a git submodule
> git submodule add https://github.com/cl8dep/dns-sync.wiki.git docs/wiki
> git commit -m "Add wiki submodule"
> ```
>
> Then push to GitHub and enable the Wiki in repository Settings.
